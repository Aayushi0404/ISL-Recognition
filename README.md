# Indian Sign Language (ISL) Recognition System

A real-time Indian Sign Language (ISL) recognition system that uses MediaPipe hand landmarks and a deep learning model to recognize and convert sign gestures into text.

## Features

* Real-time ISL gesture recognition
* Hand landmark detection using MediaPipe
* Deep learning-based gesture classification
* Live text generation from recognized signs
* Performance evaluation and training visualization

## Dataset

This project uses the Indian Sign Language (ISL) dataset by Prathum Arikeri from Kaggle and MediaPipe-based hand landmark extraction for model training and prediction.

Dataset: https://www.kaggle.com/prathumarikeri/american-sign-language-09az

## Tech Stack

* Python
* TensorFlow / Keras
* MediaPipe
* OpenCV
* NumPy
* Matplotlib

## Main Files

* `final_pred_isl_project.py` – Main ISL recognition application
* `final_pred_isl_exp.py` – Experimental version
* `final_pred_isl_basic.py` – Basic version
* `train_isl_model.py` – Model training
* `evaluate_performance.py` – Performance evaluation

## Run

python final_pred_isl_project.py
